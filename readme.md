## Mssql Test-Fixtures
========================

Just storing the results of tests against certain popular mssql node modules

The following modules are currently under tests

| Name      |  Version Tested   |
|-----------|:------:|
| [seriate](https://github.com/LeanKit-Labs/seriate)   | 0.3.1
| [mssql](https://www.npmjs.org/package/mssql)     | 1.1.0


After pulling in dependencies via `npm install`, run `gulp` to run all tests in the repo.

**NOTE:** *You must specify any database connection info `tests/config.js`.*
